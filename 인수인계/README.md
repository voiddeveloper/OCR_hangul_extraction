### 인계자 : OD팀 ( 황준환, 김종영, 신동휘 )

```
인계내용 : 
- 어떤 목표을 가지고 진행하였는가?
- 코드 파일 설명
- ocr 연구를 하면서 느낀 경험담
```

#### 1. 우리팀의 ocr 시작 목표 :
```
해외에 인기있는 게임의 자막을 우리나라 말로 실시간 번역하여,
게임을 즐기는 사용자들을 대상으로 서비스를 하고자 하였다. 
```

#### 2. 코드 파일 간단설명 :
```

각 파일에 들어가면 자세한 설명이 작성되어있다.

1. hsv_filter.py : 
 - 원하는 hsv 색상 대역만을 뽑아낸후, 
   해당 색상 대역에서 글자스러움 필터 조건을 거친 후 글자를 찾아내는 코드

2. 가 찾기.py :
 - 가','나' 등 받침이 없으며, 모음이 오른쪽에 붙어있는 글자만을 찾는 코드

3. 맑은 고딕 폰트 찾기.py :
 - 맑은 고딕 폰트의 자음, 모음만을 찾는 코드

4. similarColorBinding.py :
 - 이미지가 표현할 수 있는 색의 갯수를 줄이는 함수

5. pngTojpg.py :
 - 폴더에 있는 png파일을 jpg파일로 확장자를 바꾸어 주는 함수

6. imagePreProcessing.py :
 - 이미지에서 텍스트 영역만을 추출해내기 위해 전 처리 작업을 하는 코드이다.

7. textDetection.py :
 - 글자의 가로선과 세로선을 찾은 후, 
   밝기가 비슷한 것들만 모은 후, 글자스러움을 찾아 글자의 영역을 이미지 위에 그려주는 코드.
```

### ocr 연구를 하면서 느낀점
```
동휘 : 
    ocr 연구를 약 5개월 가량 끝에 마무리를 짓게 되었다.
    완성을 시키진 못하였지만 글자의 영역을 어느정도 잡는 것으로 끝을 맺었다.
    연구라는걸 처음 시도 해보았는데,
    가장 중요하다고 생각하는것은 이 기술을 완성시킬 수 있다는 자신감과 포기하지 않는 열정인것같다.
    
    ocr 연구를 중단한 이유는 우리 기술의 발전이 너무 더뎠다는 점과 기술 개발만을 해야한다는 점이다.
    서비스를 기획하였지만, 서비스를 하기위해선 기술개발이 필요하였다. 
    애초에 기획한 내용과 너무 달라져서 흥미를 잃은 것 같다.
    
    그리고 연구를 하면서 많이 정체되었다고 생각을 했는데, 돌아보니 발전된 점이 생각보다 많았다.
    작은 것들까지 관찰하고 디테일을 잡는 법, 이미지가 무엇인가 등등 많은 것을 배운것 같다.
    
    마지막으로, 기술개발을 흥미를 가지고, 꼭 필요성을  기술개발에 미쳐있는 사람이 도전했을때 성공 할 것 같다.
 
```
