#### 1. 우리팀의 ocr 시작 목표 :
```
해외에 인기있는 게임의 자막을 우리나라 말로 실시간 번역하여,
게임을 즐기는 사용자들을 대상으로 서비스를 하고자 하였다. 
```

#### 2. 코드 파일 간단설명 :
```

각 파일에 들어가면 자세한 설명이 작성되어있다.

1. hsv_filter.py : 
 - 원하는 hsv 색상 대역만을 뽑아낸후, 
   해당 색상 대역에서 글자스러움 필터 조건을 거친 후 글자를 찾아내는 코드

2. 가 찾기.py :
 - 가','나' 등 받침이 없으며, 모음이 오른쪽에 붙어있는 글자만을 찾는 코드

3. 맑은 고딕 폰트 찾기.py :
 - 맑은 고딕 폰트의 자음, 모음만을 찾는 코드

4. similarColorBinding.py :
 - 이미지가 표현할 수 있는 색의 갯수를 줄이는 함수

5. pngTojpg.py :
 - 폴더에 있는 png파일을 jpg파일로 확장자를 바꾸어 주는 함수

6. imagePreProcessing.py :
 - 이미지에서 텍스트 영역만을 추출해내기 위해 전 처리 작업을 하는 코드이다.

7. textDetection.py :
 - 글자의 가로선과 세로선을 찾은 후, 
   밝기가 비슷한 것들만 모은 후, 글자스러움을 찾아 글자의 영역을 이미지 위에 그려주는 코드.
   
8. font_to_image_tool.py
 - 폰트파일을 글자별로 이미지화 시키는 테스트 툴
 
9. ocr_test_tool.py
 - 이미지를 업로드해서 kakao,google,tesseract 를 쉽게 사용할 수 있게 해주는 테스트 툴
 
10. 네모찾기.py
 - ocr초반에 진행했던 네모 찾는 알고리즘
```
